# WorkshopDB

> 📌 Project made for the course **"Programming on the Java Virtual Machine"**
> **Student:** Mario Bariša, academic year 25/26 @ FIPU / UNIPU
>
>  **Status: Work in Progress (WIP)**

## About

WorkshopDB is a Spring Boot application designed to help mechanics manage their everyday work — keeping track of clients, their vehicles, and repair history.

## Features

- Add and manage clients
- Add vehicles (cars) linked to clients
- View repair history per vehicle
- Table view of all vehicles in the system

## Tech stack

- **Java 21**
- **Spring Boot 3.x**
- **Maven** — build tool
- **MySQL** — database

## Running the project

### Prerequisites

- Java 21 (JDK)
- Maven
- MySQL server (local or remote)
